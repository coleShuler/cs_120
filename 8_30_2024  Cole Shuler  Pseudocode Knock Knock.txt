Pseudocode for knock knock joke

Ask them for there name

Ask them if they would like to hear a joke

If yes, ask "knock knock". 
if not, end code with "that sucks"

The player responds with "who's there?". then ask "Hoo..."

The player responds with "Hoo who?" then the code tells the player "Why do you sound like an owl?"

Tf the player quits half way through, the game tells the player "Ok, bye bye."
